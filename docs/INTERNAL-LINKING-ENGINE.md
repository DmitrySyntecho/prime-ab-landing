# INTERNAL LINKING ENGINE

## Purpose
The Internal Linking Engine drives topical authority and conversion by systematically connecting all 262 pages through a hierarchical linking strategy. This maximizes SEO value, distributes page authority, and creates natural user journeys toward conversion forms.

---

## Core Principles

1. **Bidirectional Links** - Every link relationship goes both directions (service pages link to industries AND industries link back to services)
2. **Topical Clustering** - Links follow topic relevance, not arbitrary cross-linking
3. **Link Budget** - Each page has a maximum number of outbound links to preserve page authority
4. **Anchor Text Strategy** - Links use keyword-rich, descriptive anchor text (not "click here")
5. **Conversion Paths** - All link paths eventually funnel toward quote/contact CTAs

---

## Link Types

### 1. VERTICAL LINKS (Primary Category → Related Category)
- Service pages link down to: Industries, Locations, Packages, Resources
- Industry pages link to: Services, Locations, Resources, Comparisons
- Location pages link to: Services, Industries, Equipment, Packages
- Resource pages link to: Services, Industries, Equipment, Comparisons

### 2. HORIZONTAL LINKS (Same Category)
- Service-to-Service: Related equipment (e.g., "sound-system-rental" → "speaker-rental" sub-cluster)
- Industry-to-Industry: Related verticals (e.g., "trade-shows" → "pop-up-events")
- Location-to-Location: Geographic proximity (e.g., Miami → Tampa)
- Resource-to-Resource: Topic relevance (e.g., "led-wall-cost" → "led-wall-sizing-guide")

### 3. LATERAL LINKS (Cross-Cluster)
- Comparisons: Always link to relevant Service or Industry pages
- Packages: Link to Service, Industry, Location bundle relevance
- Equipment: Links to Services using that equipment, Locations with availability
- Tools: Links to Resources and Packages for context

## Anchor Text Strategy

✅ **GOOD** (Keyword-rich, descriptive)
- "led wall rental services" (links to /services/led-wall-rental)
- "AV production for conferences" (links to /industries/conferences)
- "Miami event lighting" (links to /locations/florida/miami)
- "how to size a sound system" (links to /resources/sound-system-sizing-guide)

❌ **BAD** (Generic, non-SEO)
- "click here"
- "learn more"
- "see more details"
- "link"
