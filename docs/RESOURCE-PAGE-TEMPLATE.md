## 12-Section Content Structure

Every resource page must include these sections in order:

### 1. SEO Title & H1
- Clear, search-intent matching headline
- Example: "LED Wall Rental Cost Guide"
- Used for page title, meta tags, and H1

### 2. Quick Answer Intro
- First paragraph directly answers the question
- Example: "LED wall rental costs range $2,000–$15,000 depending on screen size, pixel pitch, duration, and installation complexity"
- **AI Optimization**: Helps search engines extract concise answers

### 3. Key Factors Section
- Bulleted list of 5-8 factors affecting the topic
- Example factors for LED walls: screen size, pixel pitch, indoor vs outdoor, rigging requirements, event duration
- Provides scannable overview

### 4. Use Case Examples
- 4-5 real-world scenarios showing practical application
- Examples: corporate conferences, trade shows, fashion shows, experiential marketing, product launches
- Demonstrates relevance across industries

### 5. Planning Advice
- 3-5 practical tips for event planning
- Example topics: choosing screen size, working with vendors, budgeting, timing
- Positions Prime Line AV as expert resource

### 6. Related Services Links
- Link to 2-4 relevant service pages
- Examples: /services/led-wall-rental, /services/av-production, /services/sound-system-rental
- **Internal Linking**: Drives authority to service pages

### 7. Related Industries Links
- Link to 1-2 relevant industry pages
- Examples: /industries/corporate-events, /industries/trade-shows
- **Internal Linking**: Connects to commercial intent pages

### 8. Related Locations Links
- Optionally link to 1-3 city pages (if relevant to resource topic)
- Examples: /locations/miami/av-production, /locations/los-angeles/av-production
- **Internal Linking**: Geographic signal for local searches

### 9. Related Articles Section
- Link to 2-4 other resource pages
- Examples: "LED wall vs projection", "How big should an LED wall be?", "AV production cost guide"
- **Internal Linking**: Builds topical authority cluster

### 10. FAQ Section
- Minimum 5 structured Q&A pairs
- Example FAQs:
  - How much does LED wall rental cost?
  - What size LED wall do I need?
  - Do LED walls work outdoors?
  - How early should AV production be booked?
  - Do LED walls include technicians?
- **AI Optimization**: FAQs are heavily indexed by search engines

### 11. Summary / Key Takeaways
- Bullet-point summary of main article points
- Reinforces key concepts
- **AI Optimization**: Helps AI systems extract summarized information

### 12. Final CTA
- Call-to-action button/link
- Examples: "Request Quote", "Talk to a Producer", "Plan Your Event"
- Links to relevant service page or contact form

| Metric | Requirement |
|--------|-------------|
| **Word Count** | 1,200-1,800 words (concise, not padded) |
| **Paragraphs** | 3-5 sentences maximum |
| **Headings** | H2 for main sections, H3 for subsections |
| **Bullet Points** | Use liberally for scannability |
| **Technical Terms** | Define on first use |
| **Examples** | Include 8-12 real-world examples total |
| **FAQ Items** | Minimum 5 pairs (10 entries) |

## Mandatory Internal Linking Rules

Each resource page MUST link to:

- **2+ Service Pages** (e.g., /services/led-wall-rental, /services/av-production)
- **1+ Industry Page** (e.g., /industries/corporate-events)
- **1+ Location Page** (optional but recommended; e.g., /locations/miami/av-production)
- **2+ Other Resource Pages** (e.g., related guides in /resources)

**Total Minimum Internal Links: 6-8 per page**

### Linking Strategy:
- Services: Appear in "Related Services" section + woven into planning advice
- Industries: Appear in "Related Industries" section + reference use case examples
- Locations: Appear in "Related Locations" section (optional)
- Resources: Appear in "Related Articles" section + FAQ cross-references

---

## AI Optimization Rules

Resource pages must follow these rules to rank in AI search engines (OpenAI search, Perplexity, Claude, etc.):

1. **Answer Immediately** - First paragraph directly answers the headline question
2. **Use Descriptive Headings** - H2/H3 headings must describe section content (not generic "Overview")
3. **Keep Paragraphs Concise** - 3-5 sentences max; one idea per paragraph
4. **Define Technical Terms** - Define "pixel pitch", "rigging", "lumen output" on first use
5. **Include Clear Examples** - Real-world scenarios, pricing examples, timeline examples
6. **Use Lists** - Bullet points and numbered lists for scannability
7. **Structured FAQ** - Q&A format is heavily indexed by LLMs
8. **Use Keywords Naturally** - Include topic keyword 5-8 times in natural context (not keyword stuffing)

## Content Template Example: LED Wall Rental Cost

**Title**: LED Wall Rental Cost Guide

**Section 1 - Quick Answer Intro**
LED wall rental costs range from $2,000 to $15,000+ depending on screen size, pixel pitch, venue type, installation complexity, and rental duration. A typical 10x6ft indoor LED screen rental runs $3,000–$5,000 for 1-2 days, while large outdoor installations can exceed $20,000.

**Section 2 - Key Factors**
- Screen size (measured in square feet)
- Pixel pitch (resolution; lower = more expensive)
- Indoor vs outdoor installation
- Duration (day rental vs multi-day vs permanent)
- Rigging/structural requirements
- Technical operator fees
- Content creation & design
