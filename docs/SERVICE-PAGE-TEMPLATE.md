# SERVICE PAGE TEMPLATE - Prime Line AV

This document defines the **standard content structure** for all service pages on Prime Line AV.

**Purpose:** Ensure consistency, SEO optimization, and internal linking authority across all 260+ service-related pages.

---

## PAGE URL PATTERN

Each service page uses this URL structure:

```
/services/[service-name]
/rentals/[equipment-category]
```

Examples:
- `/services/av-production`
- `/services/led-wall-production`
- `/rentals/led-wall-rentals`
- `/rentals/sound-system-rentals`

---

## REQUIRED SECTIONS (14 TOTAL)

### 1. HERO SECTION
**Content:** H1 title + short description + dual CTAs

- **H1 Format:** `[Equipment/Service] for [Event Type]` or `Professional [Service Name]`
- **Description:** 1-2 sentences explaining what the service delivers
- **CTAs:** "Request Quote" + "Check Availability"
- **SEO Value:** Targets primary keyword in H1 tag

Example:
```
LED Wall Rental for Events & Conferences

Professional LED display solutions for events of any size.
From intimate conferences to massive festivals.
```

---

### 2. SERVICE OVERVIEW
**Content:** 2-3 paragraph explanation + bulleted features

- **Paragraph 1:** What the service includes
- **Paragraph 2:** Benefits and outcomes
- **Features List:** 5-7 key components or capabilities
- **SEO Value:** Targets secondary keywords (LED display types, sound coverage, etc.)

Example for LED walls:
- High-resolution LED panels (1.56mm - 10mm pixel pitch)
- Professional processing systems
- Rigging and mounting infrastructure
- Installation and setup
- 24/7 technical operation
- Breakdown and removal

---

### 3. COMMON EVENT PROBLEMS (PAIN POINTS)
**Content:** 3-5 problems this service solves

**Purpose:** Build trust, establish relevance, capture problem-based search queries

Examples:
- Poor visibility for audiences >500 people
- Weak audio coverage in large venues
- Equipment failures during critical moments
- Limited production quality for brand activations
- Complex event logistics with unreliable vendors

**SEO Value:** Targets "how to..." and "problem + solution" keywords

---

### 4. SERVICE OPTIONS / VARIATIONS
**Content:** 2-4 different configurations or approaches

**Purpose:** Show versatility, address different event needs and budgets

Example for LED walls:
- Indoor rental systems (P2.5, P3, P4 pitch)
- Outdoor rental systems (P4, P5, P10 pitch)
- Mobile LED trailers (self-contained units)
- High-resolution displays (P1.56mm for close viewing)
- Festival-scale installations (massive outdoor events)

Example for sound systems:
- Premium line array systems (5,000+ person venues)
- Corporate event PA systems (500-2,000 person events)
- Intimate venue systems (100-500 person events)
- Wireless microphone solutions (corporate presentations)
- DJ/music-focused equipment packages

---

### 5. RELATED EQUIPMENT
**Content:** Grid or list of specific equipment brands/models used in this service

**Purpose:** Link to `/equipment` pages and capture B2B searches

**Internal Linking:** Each item links to dedicated equipment page

Example for LED walls:
- ROE Visual LED Panels
- Absen LED Display Systems
- NovaStar LED Processors
- ETC Truss Rigging Systems
- For Sound Systems:
- L-Acoustics Line Array Systems
- QSC Professional Speakers
- Shure Wireless Microphone Systems
- Behringer/Soundcraft Mixing Consoles

**SEO Value:** Targets brand + "rental" searches (long-tail equipment keywords)

---

### 6. EVENT TYPES SERVED
**Content:** List of 5-7 event categories this service works for

**Purpose:** Link to `/industries` pages and establish relevance across use cases

**Internal Linking:** Each event type links to industry-specific page

Examples:
- Corporate Conferences & Meetings
- Product Launches
- Trade Shows & Expos
- Fashion Shows
- Awards Ceremonies & Galas
- Experiential Marketing Activations
- Live Concert & Festival Production

**SEO Value:** Targets "AV rental for [event type]" queries

---

### 7. LOCATIONS SERVED
**Content:** 6-10 major metropolitan areas

**Purpose:** Link to `/locations` pages and establish geographic authority

**Internal Linking:** Each location links to city/region-specific page

Format:
```
AV Production in [City]
LED Wall Rental in [City]
Sound System Rental in [City]
```

Examples:
- Los Angeles, CA
- Miami, FL
- New York, NY
- Las Vegas, NV
- Chicago, IL
- San Francisco, CA
- Atlanta, GA
- Denver, CO
- Phoenix, AZ
- Dallas, TX

**SEO Value:** Targets "AV rental near me" and geo-specific searches

---

### 8. HOW THE SERVICE WORKS
**Content:** Step-by-step process breakdown (5-7 steps)

**Purpose:** Educate prospects, build confidence, establish process authority

Example steps:
1. **Planning & Consultation** - Understand event requirements
2. **System Design** - Create custom AV solution blueprint
3. **Equipment Preparation** - Source and test all equipment
4. **Delivery & Setup** - Transport and install on-site
5. **Testing & Rehearsal** - Verify all systems and train operators
6. **Event Operation** - Provide live technical support
7. **Breakdown & Removal** - Professional strike and equipment return

**SEO Value:** Targets "how to plan AV for event" and process-based queries

---

### 9. RECOMMENDED PACKAGES
**Content:** 2-4 bundled solutions specific to this service

**Purpose:** Show pre-built solutions, simplify purchasing, link to `/packages` pages

**Internal Linking:** Each package links to dedicated package page

Examples for LED Wall:
- **Conference AV Package** - LED + sound + lighting combo
- **Fashion Show Production Package** - Full LED + technical operation
- **Festival Production Package** - Massive outdoor LED installation
- **Hybrid Event Package** - In-venue LED + streaming infrastructure

Examples for Sound Systems:
- **Corporate Meeting Package** - Wireless mics + main PA system
- **Live Music Package** - Full sound system + mixing console
- **Large Venue Package** - Line array system for 5,000+ guests

**SEO Value:** Targets "AV package for [event type]" searches

---

### 10. RELATED RESOURCES
**Content:** 5-7 educational articles/guides related to this service

**Purpose:** Link to `/resources` pages, capture "how to" and "guide" searches

**Internal Linking:** Each resource links to educational content page

Examples for LED walls:
- "How Big Should Your Event LED Wall Be? Size Guide"
- "LED Wall Rental Cost: Pricing Breakdown & Budget Guide"
- "Indoor vs Outdoor LED Displays: Complete Comparison"
- "LED Wall Pixel Pitch Explained: Choosing the Right Resolution"
- "LED Display Troubleshooting: Common Issues & Solutions"

Examples for Sound Systems:
- "Event Sound System Planning: Complete Guide"
- "Wireless Microphone Setup for Corporate Events"
- "How Many Speakers Do You Need? Sound Coverage Guide"
- "Line Array vs Traditional Speakers: Comparison"

**SEO Value:** Targets educational keywords and establishes thought leadership

---

### 11. COMPARISON LINKS
**Content:** 2-3 direct competitor/alternative comparisons

**Purpose:** Link to `/comparisons` pages, capture "vs" and decision-making searches

**Internal Linking:** Each comparison links to dedicated comparison page

Examples:
- "LED Wall vs Projection: Which is Better?"
- "Full AV Production vs Equipment-Only Rental"
- "Line Array vs Passive Speakers for Events"
- "In-House AV Team vs AV Rental Company"

**SEO Value:** Targets high-intent "X vs Y" decision queries

---

### 12. GALLERY / VISUAL PROOF
**Content:** 6-12 professional images showing service in action

**Purpose:** Build credibility, showcase real installations, improve engagement

**Image Types:**
- Installation in progress photos
- Full event setups from past clients
- Equipment detail shots
- Different venue types (indoor, outdoor, unique spaces)
- Team setup/operation photos

**Alt Text:** Include descriptive alt text for SEO and accessibility

**SEO Value:** Image search visibility + user engagement signals

---

### 13. FAQ SECTION
**Content:** 8-12 frequently asked questions specific to this service

**Purpose:** Answer specific prospects questions, target long-tail keywords

**Question Categories:**

Pricing/Budget:
- "How much does [service] rental cost?"
- "What's included in the price?"
- "Do you offer discounts for multi-day rentals?"

Specifications:
- "What specifications does your [equipment] have?"
- "What size/capacity options are available?"
- "Is [feature] included?"

Logistics:
- "Do you provide installation?"
- "Can you deliver to [location]?"
- "What's your setup timeline?"

Support:
- "Do you provide technical support during the event?"
- "What happens if equipment fails?"
- "Can you provide operators/technicians?"

Planning:
- "How far in advance should I book?"
- "Can I add/remove equipment last-minute?"
- "Do you work with event planners/coordinators?"

**SEO Value:** Targets question-based searches and featured snippet opportunities

---

### 14. FINAL CTA + LEAD FORM
**Content:** Strong call-to-action + contact form

**Purpose:** Convert interested prospects into qualified leads

**CTA Options:**
- "Request a Quote"
- "Talk to a Producer"
- "Plan My Event"
- "Schedule a Consultation"

**Form Fields (Minimum):**
- Name
- Email
- Phone
- Event Date
- Event Type
- Location
- Message/Notes
- [Service-specific field] - e.g., "LED Wall Size Needed", "Attendee Count", etc.

**SEO Value:** Conversion tracking + lead quality signals

## INTERNAL LINKING REQUIREMENTS

**Every service page MUST include links to:**

### Tier 1 (Required - minimum 2 each):
- Industry pages: 2+ links to `/industries/[event-type]`
- Location pages: 2+ links to `/locations/[city]`
- Resource pages: 2+ links to `/resources/[guide-name]`

### Tier 2 (Recommended):
- Package pages: 1+ link to `/packages/[package-name]`
- Comparison pages: 1+ link to `/comparisons/[vs-comparison]`
- Equipment pages: 2-5 links to `/equipment/[brand-model]`

### Tier 3 (Cross-selling):
- Related service pages: 1-2 links to `/services/[related-service]`
- Homepage category links: 1 link back to `/services` hub

**Total Internal Links Per Page:** 12-20 outgoing links minimum

- **Pain Points:** 150-200 words
- **Service Options:** 200-300 words
- **Equipment Section:** 100-150 words
- **How It Works:** 250-350 words
- **Resources Section:** 150-200 words
- **FAQ Answers:** 50-100 words each

**Total Page Content:** 1,800-2,400 words (optimal for SEO)

## IMPLEMENTATION PRIORITY

1. **Tier 1 (Critical):** 15 main service pages (LED, Sound, Lighting, Staging, etc.)
2. **Tier 2 (High):** 20-30 equipment-specific rental pages
3. **Tier 3 (Medium):** 40-50 niche service variations
4. **Tier 4 (Low):** 100+ equipment-detail pages

Each tier tier should be completed before moving to the next.
